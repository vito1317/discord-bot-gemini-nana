# discord-bot-gemini-nana
discord-bot-gemini-nana
利用gemini-flash-1.5模型製作的奈奈Ai(心理支持輔導) discord機器人

Nana Ai (psychological support and counseling) discord robot made using gemini-flash-1.5 model

Nana Ai is a Discord bot powered by Gemini-flash-1.5 that provides on-demand emotional support and counseling. Whether you're feeling stressed, anxious, or just need someone to talk to, Nana Ai is here to help.
